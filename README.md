# Black Angler

Welcome to the Black Angler project.

## Directory Structure
- `public/` : Static files served directly
  - `index.html` : Main entry point
  - `css/` : Stylesheets
  - `js/` : Scripts
  - `images/` : Image assets
  - `assets/` : Other assets (fonts, etc.)
- `src/` : Source files and templates
  - `pages/` : Page templates
  - `components/` : Reusable components
  - `layouts/` : Layout structures

## Design Philosophy
This project uses a dark, premium aesthetic with smooth gradients, glassmorphism, and dynamic animations.
